# Use Cases - Wymagania Ogolne

Wymagania funkcjonalne ogolne dla aplikacji WaterLife.

---

## Use Case 1: Responsywnosc (Mobile/Tablet)
**Status: DO SPRAWDZENIA**

Strona odpowiednio przystosowana do obslugi z telefonu komorkowego (mobile) lub tableta.

**Cel:** Dostepnosc aplikacji na wszystkich urzadzeniach.

---

## Use Case 2: Ikonka logowania
**Status: ZAIMPLEMENTOWANE**

Obsluga ikonki do logowania na stronie glownej.
- Dropdown z opcjami "Zaloguj sie" / "Zaloz konto"
- Modal logowania/rejestracji
- Po zalogowaniu: inicjaly uzytkownika, opcja "Wyloguj"

**Cel:** Umozliwienie klientom tworzenia kont i logowania.

---

## Use Case 3: Wyszukiwarka w header
**Status: ZAIMPLEMENTOWANE**

Wyszukiwarka w header dla strony glownej.
- Autocomplete z sugestiami produktow
- Przekierowanie do /produkty?search=query
- Ukryta na stronie /produkty (ma dedykowana wyszukiwarke)

**Cel:** Globalne wyszukiwanie produktow z kazdej strony.
